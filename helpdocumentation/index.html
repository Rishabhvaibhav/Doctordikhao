Setup Admin Panel #back to top

Admin panel or backend is require to upload quote, images and videos which will be reflected in your android application, you require a linux based server with php support, if you are not aware of this ask your server provider about configuration.

For content based project like ours, we recommend using VPS server to speed up content loading, while you can also use shared server for startup

Free server will not work with this script, as we are using web API and free server provide advertising with API call


You can watch installation video below this steps

  • 1. Unzip downloaded zip from code canyon.
  • 2. FOR MAC OS USERS ONLY : windows user go to Step 3
    • 2.1 : Under Folder Website --> PHPScript check if .env file is visible ?
    • 2.2 : if Not visible than you can make it visible by pressing COMMAND + SHIFT + (.DOT)


  • 3. Zip Php Script to upload
    • 2.2 : Under Folder Website --> PHPScript Select all the files
    • 2.3 : Right Click and compress in zip file


  • 4. In Cpanel Click onFile Manager Open a folder where you want to upload --> select upload option --> Upload Zip file


  • 5. Now Unzip/Extract zip file on server


  • 6. Setup Dataabase --> Go to MySQL® Databases --> Create New Database


  • 7. On same page you need to Create New Database User
    • createuser

  • 8. Add User to Database --> 1. select your database --> 2. select Database user --> and Add
    It will open new page for privileges, here click on ALL PRIVILEGES and submit


  • 9. Now Open phpmyadmin --> Upload database file


  • 10. Update .env file
    • 10.1 : In phpMyAdmin edit file .env right click on file and select Edit
    • 10.2 : Change DB_DATABASE Add Database name we created in step 6
    • 10.3 : Change DB_USERNAME and DB_PASSWORD username and password we created in step 7

Setup Realtime Database#back to top

  • 1. Open firebase console and select Firestore Database Tab from left side menu bar.
  • 2. Create Database button click.

  • 3. Select option United States in Database location,

  • 4. Select option 1 Start in locked mode,

  • 5. In rules tab update rules as shown in below and click publish button.
    •                                             // [your package name here]   for example companyname.projectname
                                            ".read": true, 
                                            ".write": true


Setup Cloud Firestore Database#back to top

  • 1. Open firebase console and select Firestore Database Tab from left side menu bar.

  • 2. Select start in production mode and press Create Database button.


  • 3. Select option United States in Database location,


  • 4. Select United States in database location and press enable button.


  • 5. In rules tab update rules as shown in below and click publish button.
    •                                             // [your package name here]   for example companyname.projectname
                                            service cloud.firestore {
                                                match /databases/{database}/documents { match /{document=**} {
                                                allow read: if true;
                                                allow write: if true; }
                                                } 
                                            }

Setup ConnectCube Video Calling#back to top

    1. Go to connectycube.com https://admin.connectycube.com/signin and login or register there

    2. You can create new app and get Applicatio id, authorization key and Application secreat

    3. In Android Studio and open file lib/VideoCall/utils/configs.dart and update details here.

    4. Also change connectcube details in server in Controller.php file
    file path is "/Controllers/Controller.php"

    5. Configure connectycube with firebase- get server key from firebase and setup in connectycube

    6. add copied server key in connectycube

    7. Also Change Server token in lib/main.dart serverToken = “Your firebase Server Key”

Release JKS file not found#back to top

  • 1. Open android->key.properties in Android Studio
  • 2. Find release.jks file in project, get path of this file
  • 3. Change path in storefin in key.properties file
    •                                     storePassword=android
                                    keyPassword=android
                                    keyAlias=key0
                                    storeFile=/Volumes/Development/doctorfinder_timeslot-main/relese.jks

Create Map API Key#back to top

  • 1. Go to the Google Cloud Platform Console. Click Here

  • 2. Create project



  • 3. Select APIs & Services > library. Enable Maps javascript api, Maps Embed API, Places API, Geocoding API


  • 4. In side menu click on Keys & Credentials.



  • 5. You can restrict map api key by adding website or app details



  • 6. Copy API key and set in .env file on your server
